## Restaurants microservice

Based in framework [Nest](https://github.com/nestjs/nest) 

## First the first

Install dependecies with the next command in your CLI
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
